### Outlines for Detecting Ambushes

1. **Using CHAT_MSG_SYSTEM Event**:
    - The World of Warcraft API allows monitoring of system messages sent to the chat by registering for the `CHAT_MSG_SYSTEM` event.
    - A handler can capture messages and match patterns or specific keywords related to ambush notifications.
    - For example, the event handler can look for text like 'Ambush imminent!' or similar phrases that appear in the system message.

2. **Alternative: Monitoring Combat Log Events**:
    - The `COMBAT_LOG_EVENT_UNFILTERED` event could offer another way to detect ambushes if the system provides relevant combat log entries.
    - This involves parsing the combat log for specific entries or spell IDs tied to ambushes.
    - Example: Scanning for an event subtype (like `SPELL_CAST_START`) and checking the associated spell/ability name or ID.

These outlines can be added to the `pd` inspection logic for testing and refinement.
